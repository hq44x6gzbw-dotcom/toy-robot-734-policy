1. Overview

The Toy Robot 734 API is an experimental service that generates introspective narrative text based on simulated robot behavior. This service is powered by Google's Gemini AI models and does not collect, store, or share any personal user data.

2. Data Collection

We do not collect or store:

  • Personal identification information (names, emails, IP addresses)
  • User account data or authentication credentials
  • Conversation history or API request logs
  • Cookies, tracking pixels, or analytics data

3. Data Processing

All API requests are processed in real-time and discarded immediately after generating a response. The simulation state (robot position, orientation, and movement history) exists only for the duration of a single API call.

When you interact with the Toy Robot 734 Custom GPT, OpenAI may process your conversation according to their own privacy policy. We recommend reviewing OpenAI's Privacy Policy for details.

4. Third-Party Services

This API uses Google's Gemini AI to generate narrative responses. Google may process API requests according to their own terms. We recommend reviewing Google's Gemini API Terms for more information.

5. Data Retention

We do not retain any data. All simulation states and generated narratives are ephemeral and exist only in memory during request processing.

6. Children's Privacy

This service is not directed at children under 13 years of age. We do not knowingly collect personal information from children.

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated "Last Updated" date.

8. Contact Information

For questions about this privacy policy, please contact the project maintainer via the GitHub repository.
